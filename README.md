# NonEuclid Mansion
A game about wondering through a constantly changing mansion.
Basied off @hackerpoet's NonEuclidean game engine

## Source Code Dependencies
Add glew-2.1.0 to the main directory

## Controls
* **Mouse** - Look around
* **AWSD** - Movement
* **Alt + Enter** - Toggle Fullscreen
* **Esc** - Exit demo
